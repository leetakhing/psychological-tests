
**GitHub**: https://github.com/leetakhing/psychological-tests

**JSON 格式**：
```json
{
  "id": "test-id",
  "title": "标题",
  "questions": [{"question": "...", "options": [...]}],
  "results": {"typeA": {"title": "...", "description": "..."}}
}
```

**使用方式**：
1. 本地: `python -m http.server 8000`
2. 在线: GitHub Pages / Vercel / Netlify（全部免费）
